Linkchecker Context Repo.
=================


#Usage

Command structure:

  linux / iOS:

```
    docker run -v $(pwd):/data praqma/linkchecker:v[version number] [Options] [Output Options] [URL]
```

  Windows:

```
    docker run -v /$(pwd):\\data praqma/linkchecker:v[version number] [Options] [Output Options] [URL]
```
